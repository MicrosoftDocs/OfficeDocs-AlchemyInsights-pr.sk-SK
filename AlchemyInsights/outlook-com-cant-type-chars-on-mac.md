# <a name="how-to-type-special-characters-on-a-mac"></a>Ako písať špeciálne znaky na Macu

Ak chcete do Macu zadávať špeciálne znaky, otvorte ponuku **špeciálne znaky** pomocou klávesovej skratky ⌘ **Command** + **Control** + **Space**.

Rýchla skratka pre @ je tiež **ALT** + **64**.
