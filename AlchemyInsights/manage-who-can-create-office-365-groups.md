# <a name="manage-who-can-create-microsoft-365-groups"></a>Spravovanie osôb, ktoré môžu vytvárať skupiny Microsoft 365

V predvolenom nastavení môže ktokoľvek vo vašej organizácii vytvoriť skupiny služby O365.
  
Ak chcete zmeniť, môžete vytvoriť skupinu zabezpečenia s PowerShell, ktorý obmedzuje vytváranie skupiny na podmnožinu používateľov.
  
Postupujte podľa krok-za-krokom návod tu: [Správa, kto môže vytvoriť Microsoft 365 skupiny](https://docs.microsoft.com/office365/admin/create-groups/manage-creation-of-groups)
