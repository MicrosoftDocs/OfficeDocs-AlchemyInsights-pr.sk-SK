# <a name="onedrive-common-issues-and-resolutions"></a>Bežné problémy a riešenia vo OneDrive

Tu sú niektoré často požadované témy týkajúce sa problémov s OneDrivom:

- **Nastavenie synchronizácie onedrivu**: [Synchronizujte súbory OneDrivu for Business](https://go.microsoft.com/fwlink/?linkid=533375) alebo [synchronizujte súbory SharePointu](https://go.microsoft.com/fwlink/?linkid=871666) s klientom synchronizácie OneDrivu pre Windows.  Používate Mac? Súbory s [OneDrivom](https://support.office.com/article/Sync-files-with-the-OneDrive-sync-client-on-Mac-OS-X-d11b9f29-00bb-4172-be39-997da46f913f) môžete synchronizovať aj s ním.

- **Riešenie problémov so synchronizáciou OneDrivu**: Ak OneDrive nesynchronizuje súbory podľa očakávania, môžete [riešiť problémy so synchronizáciou OneDrivu](https://go.microsoft.com/fwlink/?linkid=866431) pre Windows. Používate Mac? Môžete [riešiť aj problémy so synchronizáciou.](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756)
- **Obmedzenia onedrivu**: Získajte informácie o [obmedzeniach a obmedzeniach,](https://support.office.com/article/Invalid-file-names-and-file-types-in-OneDrive-OneDrive-for-Business-and-SharePoint-64883a5d-228e-48f5-b3d2-eb39e07630fa) ktoré sa pri synchronizácii so službou OneDrive vzťahujú na názvy súborov a typy súborov.
- **Úložisko vo OneDrive**: Zistite, ako [zmeniť ukladací priestor vo OneDrive konkrétneho používateľa](https://docs.microsoft.com/onedrive/change-user-storage) alebo [spravovať ukladací priestor vo OneDrive for Business](https://support.office.com/article/Manage-your-OneDrive-for-Business-storage-31519161-059C-4764-B6F8-F5CD29F7FE68).
- **Denníky auditu**: Zistite, ako môžete [použiť vyhľadávanie denníka auditu](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance#search-the-audit-log) na kontrolu aktivity súborov, priečinkov, používateľov alebo povolení.
