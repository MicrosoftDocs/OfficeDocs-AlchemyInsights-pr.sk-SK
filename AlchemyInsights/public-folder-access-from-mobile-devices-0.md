# <a name="public-folder-access-from-mobile-devices"></a>Prístup k verejným priečinkom z mobilných zariadení

Verejné priečinky nie sú prístupné z mobilných zariadení.
  
Ak chcete mať funkcie podobné verejným priečinkom a prístup k nemu z mobilných zariadení, pozrite si [informácie o Microsoft 365 skupiny](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) pre lepšiu alternatívu.
