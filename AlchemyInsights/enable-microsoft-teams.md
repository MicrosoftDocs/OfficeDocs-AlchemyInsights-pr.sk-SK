# <a name="enable-and-use-microsoft-teams"></a>Povolenie a používanie tímov Microsoft teams

Ak nemáte prístup k tímom Microsoft Teams, pozrite sa, ako získam [prístup k tímom Microsoft teams](https://support.office.com/article/How-do-I-get-access-to-Microsoft-Teams-fc7f1634-abd3-4f26-a597-9df16e4ca65b.aspx).

Ak už máte tímy Microsoft a potrebujete vedieť, ako ho nakonfigurovať a spravovať, pozrite si [Správa nastavení Microsoft teams pre vašu organizáciu](https://docs.microsoft.com/MicrosoftTeams/enable-features-office-365).
