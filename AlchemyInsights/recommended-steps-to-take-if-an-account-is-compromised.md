[<span data-ttu-id="7b2d4-103">VIDEO: Oprava v prípade zneužitia konta Office 365</span><span class="sxs-lookup"><span data-stu-id="7b2d4-103">VIDEO: Fixing a compromised Office 365 account</span></span>](https://www.microsoft.com/videoplayer/embed/RE2jvOb?pid=ocpVideo0-innerdiv-oneplayer&amp;postJsllMsg=true&amp;maskLevel=20&amp;autoplay=true)
  
1. <span data-ttu-id="7b2d4-104">Ihneď [vytvorte nové heslo používateľa](https://docs.microsoft.com/office365/admin/add-users/reset-passwords).</span><span class="sxs-lookup"><span data-stu-id="7b2d4-104">[Reset the user's password](https://docs.microsoft.com/office365/admin/add-users/reset-passwords) immediately.</span></span> <span data-ttu-id="7b2d4-105">Neoznamujte nové heslo koncovému používateľovi e-mailom.</span><span class="sxs-lookup"><span data-stu-id="7b2d4-105">Do not communicate the new password through email to the end user.</span></span>

2. <span data-ttu-id="7b2d4-106">Odstráňte všetky podozrivé [adresy na preposielanie](https://docs.microsoft.com/office365/admin/email/configure-email-forwarding) nastavené na úrovni poštovej schránky.</span><span class="sxs-lookup"><span data-stu-id="7b2d4-106">Remove any suspicious [forwarding addresses](https://docs.microsoft.com/office365/admin/email/configure-email-forwarding) set at the mailbox level.</span></span>

3. <span data-ttu-id="7b2d4-107">Odstráňte všetky podozrivé [pravidlá pre doručenú poštu](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) nastavené v poštovej schránke.</span><span class="sxs-lookup"><span data-stu-id="7b2d4-107">Remove any suspicious [inbox rules](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) set within the mailbox.</span></span>

4. <span data-ttu-id="7b2d4-108">Ak má používateľ zablokované odosielanie e-mailov, [prejdite na stránku obmedzených používateľov a konto odblokujte](https://protection.office.com/?hash=/restrictedusers).</span><span class="sxs-lookup"><span data-stu-id="7b2d4-108">If the user is blocked from sending email, [go to the Restricted Users to unblock the account](https://protection.office.com/?hash=/restrictedusers).</span></span> <span data-ttu-id="7b2d4-109">Po uplynutí jednej hodiny po dokončení by mal používateľ môcť odosielať správy.</span><span class="sxs-lookup"><span data-stu-id="7b2d4-109">Once done, the user should be able to resume sending messages within 1 hour.</span></span>

5. <span data-ttu-id="7b2d4-110">Odstráňte konto používateľa z každej [skupiny rol správcu](https://docs.microsoft.com//office365/admin/add-users/assign-admin-roles), kým si nie ste istí, že konto už nie je zneužité.</span><span class="sxs-lookup"><span data-stu-id="7b2d4-110">Remove the user account from any [administrative role groups](https://docs.microsoft.com//office365/admin/add-users/assign-admin-roles) until you are confident that the account is no longer compromised.</span></span>

<span data-ttu-id="7b2d4-111">Do budúcnosti na minimalizovanie rizika úniku údajov alebo zneužitia konta odporúčame prečítať si náš [pláne zabezpečenia služieb Office 365](https://docs.microsoft.com//office365/securitycompliance/security-roadmap).</span><span class="sxs-lookup"><span data-stu-id="7b2d4-111">To minimize the potential of a data breach or a compromised account in the future, we recommend reading our [Office 365 Security best practices article](https://docs.microsoft.com//office365/securitycompliance/security-roadmap).</span></span>
