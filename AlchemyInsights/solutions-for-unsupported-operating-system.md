"Nepodporovaný operačný systém"

# <a name="solutions-for-unsupported-operating-system"></a>Riešenia pre "Nepodporovaný operačný systém"

Pri pokuse nainštalovať Office 365 2016, môže zobraziť nasledujúce chybové hlásenie, ak zariadenie nespĺňa [systémové požiadavky](https://products.office.com/office-system-requirements).
  
 **"Nepodporovaný operačný systém"**
  
Vyžaduje Office 365/2016:
  
PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 alebo Windows Server 2008 R2
  
 alebo 
  
Mac OS X 10.10
  
Ak dostal túto správu pri pokuse o inštaláciu balíka Office v mobilnom zariadení alebo Chromebook, pozrite si [Nastavenie aplikácie balíka Office a e-mailom na mobilnom zariadení](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f?wt.mc_id=Alchemy_ClientDIA.aspx).
