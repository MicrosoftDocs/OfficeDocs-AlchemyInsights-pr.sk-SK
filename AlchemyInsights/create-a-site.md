# <a name="create-a-site"></a>Vytvorenie lokality

- Prejdite na [stránku Microsoft 365 admin Center](https://portal.office.com/adminportal/home#/SitesList)a potom vyberte možnosť **pridať stránky**. 
    
- Nové centrum správy služby SharePoint na stránke lokality, kliknite na **vytvoriť**. 
    
 **Poznámka:** Nemôžete vytvoriť nový web, ktorý má rovnakú adresu URL s existujúcej lokality. 
  
Ďalšie informácie o vytváraní stránky nájdete v téme [Vytvorenie stránky](https://go.microsoft.com/fwlink/?linkid=866295).
  
Informácie o povolení používateľom vytvárať svoje vlastné stránky, nájdete v časti [Vytvorenie spravovať lokality SharePoint Online](https://go.microsoft.com/fwlink/?linkid=866296).
