# <a name="microsoft-365-multi-geo"></a>Microsoft 365 multi-Geo

S Microsoft 365 multi-Geo, vaša organizácia môže rozšíriť svoju prítomnosť Microsoft 365 do viacerých geografických oblastí a/alebo krajín v rámci vášho existujúceho nájomcu. Ďalšie informácie nájdete v nižšie uvedených odkazoch.

- [Microsoft 365 multi-Geo](https://docs.microsoft.com/office365/enterprise/office-365-multi-geo)
- [Plán pre Microsoft 365 multi-Geo](https://docs.microsoft.com/office365/enterprise/plan-for-multi-geo)
- [Konfigurovať multi-Geo](https://docs.microsoft.com/office365/enterprise/multi-geo-tenant-configuration)
