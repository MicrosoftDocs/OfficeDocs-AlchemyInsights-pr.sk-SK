# <a name="your-users-mailbox-is-being-created"></a>Sa vytvorí poštová schránka používateľa

Môže trvať až hodinu, kým je Nová poštová schránka pripravená na použitie.
  
Ak je poštová schránka používateľa "pripravuje" viac ako hodinu, môžete skúsiť odstrániť a znova vytvoriť používateľa, alebo použite možnosť "Zavolaj vám" hovoriť s pracovník technickej podpory.
