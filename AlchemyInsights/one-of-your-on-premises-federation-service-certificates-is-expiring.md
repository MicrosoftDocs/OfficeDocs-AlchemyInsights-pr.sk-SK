# <a name="one-of-your-on-premises-federation-service-certificates-is-expiring"></a>Jeden z vašich lokálnych certifikátov služieb federácie je končí

Ak chcete vyriešiť tento problém, postupujte nasledovne:
  
- Nainštalovať Microsoft Azure Active Directory modul pre Windows PowerShell v počítači (Ak modul nie je už nainštalovaný). Chcete urobiť, prejdite na [Azure Active Directory PowerShell pre graf](https://docs.microsoft.com/powershell/azure/active-directory/install-adv2?view=azureadps-2.0)
    
- Postupujte podľa krokov v "Scenár 1: AD FS token podpisu certifikátu skončila" časť ["Vyskytol sa problém s prístupom k lokalite" chyba AD FS, keď externý používateľ prihlási do balíka Office 365, Azure alebo Intune](https://support.microsoft.com/help/2713898/there-was-a-problem-accessing-the-site-error-from-ad-fs-when-a-federat).
    
- Postupujte podľa krokov v t[ako aktualizovať alebo opraviť nastavenia externej domény v balíku Office 365, Azure alebo Intune](https://support.microsoft.com/help/2647048/how-to-update-or-repair-the-settings-of-a-federated-domain-in-office-3).
    
Ďalšie informácie o obnovení federácie certifikátov nájdete [obnovenie certifikátov služby O365 a Azure AD](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-o365-certs).
