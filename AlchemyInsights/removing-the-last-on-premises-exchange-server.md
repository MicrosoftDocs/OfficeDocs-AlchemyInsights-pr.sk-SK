# <a name="removing-the-last-on-premises-exchange-server"></a>Odstránenie posledného lokálneho servera Exchange

Pred odstránením posledného servera Exchange z lokálnej organizácie by ste si mali uvedomiť, aké dôsledky tohto postupu vám pomôžu rozhodnúť sa, či budete potrebovať posledný server na iné účely. Ďalšie informácie nájdete [v téme ako a kedy Vyradenie lokálnych serverov Exchange v hybridnom nasadení](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
