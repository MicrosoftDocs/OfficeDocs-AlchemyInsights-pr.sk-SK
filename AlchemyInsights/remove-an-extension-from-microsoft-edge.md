# <a name="remove-an-extension-from-microsoft-edge"></a>Odstránenie rozšírenia z prehliadača Microsoft Edge

Ak chcete odstrániť rozšírenie z prehliadača Microsoft Edge:

1. Otvorte Microsoft Edge.
2. Vyberte položku **... > prípony**.
3. Na stránke **nainštalované rozšírenia** vyhľadajte príponu, ktorú chcete odstrániť, a potom vyberte položku **odstrániť** vedľa nej.
4. V potvrdzovacom dialógovom okne vyberte položku **odstrániť**.
