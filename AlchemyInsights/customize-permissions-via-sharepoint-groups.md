# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>Prispôsobenie povolení lokality SharePoint prostredníctvom skupín SharePoint 

Skupina lokality SharePoint je kolekcia používateľov, ktorí majú rovnaké množiny povolení pre lokality a obsah. Skôr než priraďte povolenia jednu osobu naraz, môžete použiť skupiny na pohodlné priraďovanie rovnakej úrovne povolení mnohým ľuďom naraz.

[Prispôsobenie povolení lokality SharePoint](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)
