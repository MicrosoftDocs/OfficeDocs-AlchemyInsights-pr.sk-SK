# <a name="create-mailboxes-that-dont-require-licenses"></a>Vytvorenie poštových schránok, ktoré nevyžadujú licencie

- Ak potrebujete skupinovú e-mailovú adresu pre viacerých ľudí, pozrite si tému [vytvorenie zdieľanej poštovej schránky](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox).

- Ďalšie informácie o tom, ako umožniť tímu ľudí spolupracovať a zdieľať e-maily a súbory, nájdete v téme [informácie o skupinách v službe Microsoft 365](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
