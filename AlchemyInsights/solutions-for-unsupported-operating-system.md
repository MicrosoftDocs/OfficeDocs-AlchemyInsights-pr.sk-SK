# <a name="solutions-for-unsupported-operating-system"></a>Riešenia pre nepodporovaný operačný systém

Pri pokuse o inštaláciu balíka Microsoft 365 sa môže zobraziť chybové hlásenie **nepodporovaného operačného systému** , ak vaše zariadenie nespĺňa [systémové požiadavky](https://products.office.com/office-system-requirements).
  
Microsoft 365 vyžaduje:
  
- PC: Windows 10, Windows 8,1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 alebo Windows Server 2008 R2

    ALEBO

- Mac OS X 10,10

Ak sa pri pokuse o inštaláciu balíka Office v mobilnom zariadení alebo Chromebooku zobrazilo Toto hlásenie, pozrite si tému [Inštalácia a spustenie balíka Microsoft Office v Chromebooku](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
