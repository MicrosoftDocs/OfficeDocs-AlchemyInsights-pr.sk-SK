# <a name="identify-your-tenant-id"></a>Identifikácia ID nájomníka

V centre spravovania služby Azure vyhľadajte svoje ID organizácie Microsoft 365 (ID nájomníka).

1. Na [stránke vlastnosti](https://aka.ms/AzurePropertiesPage)prejdite do poľa **Identifikácia adresára** .

2. ID skopírujte podľa potreby.

Ďalšie informácie nájdete v téme [Vyhľadanie ID organizácie v službe Microsoft 365](https://docs.microsoft.com/onedrive/find-your-office-365-tenant-id).
