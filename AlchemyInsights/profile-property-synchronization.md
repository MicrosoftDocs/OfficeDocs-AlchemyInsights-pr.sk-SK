# <a name="profile-property-synchronization"></a>Synchronizácia vlastností profilu

Nie je možné priradiť ďalšie vlastnosti pre import reklamy synchronizovať z Azure Active Directory aplikácie používateľského profilu. Import reklám je obmedzený na predkonfigurovanú množinu vlastností, aby sa zaručila konzistentná výkonnosť úlohy časovača.
  
[Ďalšie informácie o vlastnostiach profilu, ktoré sa synchronizujú](https://go.microsoft.com/fwlink/?linkid=875671)
