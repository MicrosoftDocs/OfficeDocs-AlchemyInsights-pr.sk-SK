# <a name="create-and-use-a-shared-mailbox"></a>Vytvorenie a používanie zdieľanej poštovej schránky

Zdieľaná poštová schránka umožňuje viac ako jednej osobe odosielať a prijímať e-maily pre alias, napríklad **sales@contoso.com**, bez priradenia inej licencie.
  
[Vytvorenie zdieľanej poštovej schránky](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) vysvetľuje, ako: 
  
- Vytvorenie zdieľanej poštovej schránky
    
- Poskytnite používateľom povolenia na používanie zdieľanej poštovej schránky
    
- Prístup k zdieľanej poštovej schránke v programe Outlook alebo v programe Outlook na webe
