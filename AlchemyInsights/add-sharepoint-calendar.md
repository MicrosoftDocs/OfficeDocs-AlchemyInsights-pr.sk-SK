# <a name="add-a-sharepoint-calendar"></a>Pridanie kalendára SharePointu

Kalendáre SharePointu sú klasické typy zoznamov, ktoré sa niekedy prelínajú s kalendármi skupiny Microsoft 365.
 
Ak chcete vytvoriť kalendár SharePointu, postupujte podľa týchto krokov:
 
1.  Vyberte ikonu ozubeného kolieska **Nastavenie** a potom vyberte položku **pridať aplikáciu**.
2.  Do vyhľadávacieho poľa **hľadanie aplikácie** zadajte text **Kalendár** a stlačte kláves ENTER.
3.  Vyberte aplikáciu s názvom **Kalendár**a potom zadajte názov kalendára.

Ďalšie informácie o skupinách v službe Microsoft 365 nájdete v téme [informácie o skupinách microsoft 365](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2).
