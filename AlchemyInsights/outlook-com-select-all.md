# <a name="how-to-select-all-in-outlookcom"></a>Ako "vybrať všetko" v službe Outlook.com

Môžete len vybrať všetko, ak všetky správy sú viditeľné. Cieľom je zabrániť náhodnému akcie na správy nemôžete vidieť.

**CTRL + kliknutie** vyberie viaceré jednotlivé správy.

**SHIFT + kliknutím** vyberie všetky správy medzi prvú a poslednú správu vyberiete.
