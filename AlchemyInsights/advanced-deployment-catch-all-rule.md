# <a name="deploy-office-apps"></a>Nasadiť Office apps

Na nasadenie aplikácie balíka Office na viacerých koncových používateľov pomocou nástroja nasadenia Office System Center Configuration Manager, ako súčasť obrazu disku operačného systému, alebo v prípade zdieľaného počítača pomocou Windows Server Remote Desktop Services (RDS) alebo terminálu Služby, zvážte použitie krok za krokom sprievodcu **Office 365 ProPlus nasadenie poradca**.
  
Než začnete, uistite sa, že ste prihlásený ako správca nájomcu alebo globálne admin a kliknite na tlačidlo tu: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
