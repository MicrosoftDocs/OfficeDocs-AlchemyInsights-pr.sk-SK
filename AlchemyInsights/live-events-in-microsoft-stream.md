# <a name="live-events-in-microsoft-stream"></a>Živé podujatia v službe Microsoft Stream

- [Vytvorenie živej udalosti v službe Microsoft Stream](https://docs.microsoft.com/stream/live-create-event)
- [Začíname pracovať s udalosťami služby Microsoft teams naživo](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
