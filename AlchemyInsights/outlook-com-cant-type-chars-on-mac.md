# <a name="how-to-type-special-characters-on-a-mac"></a>Ako písať špeciálne znaky na Mac

Zadávať špeciálne znaky v Macu, otvorte ponuku **špeciálnych znakov** s klávesovú skratku ⌘ **príkaz** + **ovládanie** + **priestoru**.

Rýchly odkaz na @ je tiež **ALT** + **64**.
