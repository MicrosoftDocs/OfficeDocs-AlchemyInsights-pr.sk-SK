# <a name="add-permissions"></a>Pridanie povolení

1. Pomocou koncového bodu súhlasu správcu [Nakonfigurujte zoznam staticky požadovaných povolení pre aplikáciu](https://docs.microsoft.com/azure/active-directory/develop/v2-permissions-and-consent#to-configure-the-list-of-statically-requested-permissions-for-an-application).
1. K dispozícii sú dve povolenia na udeľovanie povolení na vytváranie registrácií aplikácií, pričom každý z nich má odlišné správanie. Ďalšie informácie nájdete v téme [povolenia na registráciu aplikácií pre vlastné roly v službe Azure Active Directory](https://docs.microsoft.com/azure/active-directory/roles/custom-available-permissions).
1. [Povolenia roly](https://docs.microsoft.com/azure/active-directory/roles/permissions-reference#role-permissions): tabuľky, ktoré popisujú konkrétne povolenia v službe Azure Active Directory danej roly. Niektoré roly môžu mať ďalšie povolenia v službách Microsoft mimo služby Azure Active Directory.
