# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Použite SaRA diagnostikovať a vyriešiť problémy s overovaním služby Outlook

Odporúčame vám použiť [program Outlook neustále pýta na heslo](https://aka.ms/SaRA-OutlookPwdPrompt-Alchemy) diagnostické postihnutého počítača problémov kde Outlook neustále vyzve na zadanie hesla. Tento [SaRA](https://diagnostics.office.com/#/) diagnostické robí automatizované kontroly a vráti možné riešenia môžete použiť na riešenie zistených problémov.
