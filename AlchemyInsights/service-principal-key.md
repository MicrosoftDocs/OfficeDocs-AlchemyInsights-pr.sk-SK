# <a name="service-principal-key"></a>Hlavný kľúč služby

Pokyny na spravovanie kľúčových poverení pre hlavné služby nájdete v nasledujúcich článkoch:

- [Get-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/get-azureadserviceprincipalkeycredential) -získanie kľúčových poverení pre hlavné služby pomocou prostredia PowerShell.

- [New-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/new-azureadserviceprincipalkeycredential) -vytvorenie nového kľúčového poverenia pre hlavné služby pomocou prostredia PowerShell.

- [Odstrániť-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/remove-azureadserviceprincipalkeycredential) -Ak chcete odstrániť kľúčové poverenia zo služby Principal pomocou prostredia PowerShell.
