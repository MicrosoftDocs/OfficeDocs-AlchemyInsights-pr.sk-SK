# <a name="manage-who-can-create-microsoft-365-groups"></a>Spravovanie osôb, ktoré môžu vytvárať skupiny v programe Microsoft 365

Každý používateľ vo vašej organizácii môže na základe predvoleného nastavenia vytvárať skupiny so službou O365.
  
Ak chcete zmeniť, môžete vytvoriť skupinu zabezpečenia s prostredím PowerShell, ktoré obmedzuje tvorbu skupiny na podmnožinu používateľov.
  
Postupujte podľa podrobných pokynov tu: [spravovanie osôb, ktoré môžu vytvárať skupiny v programe Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
