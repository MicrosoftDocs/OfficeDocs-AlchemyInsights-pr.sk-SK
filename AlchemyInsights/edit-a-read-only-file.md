# <a name="edit-a-read-only-file"></a>Úprava súboru iba na čítanie

V knižniciach dokumentov, keď kliknete na názov dokumentu, program balíka Office môže neočakávane otvoriť ako súbor iba na čítanie. Ak ho chcete otvoriť pre úpravy, kliknite na tlačidlo **Otvoriť** (tri bodky) a potom kliknite na položku **Upraviť.**
