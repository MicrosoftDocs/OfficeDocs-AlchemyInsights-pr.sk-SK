# <a name="sending-email-as-the-shared-mailbox"></a>Posielanie e-mailov ako zdieľanej poštovej schránky

Po vytvorení zdieľanej poštovej schránky, používateľov, ktorí sú členmi poštovej schránky sú schopní poslať e-mail ako zdieľanej poštovej schránke identitu (napríklad *info@contoso.com* ). Pokyny nájdete v časti: 
  
[Otvorenie a používanie zdieľanej poštovej schránky v programe Outlook 2016 a Outlook 2013](https://support.office.com/en-us/article/open-and-use-a-shared-mailbox-in-outlook-2016-and-outlook-2013-d94a8e9e-21f1-4240-808b-de9c9c088afd)
  
[Pridať zdieľanej poštovej schránky programu Outlook na webe](https://support.office.com/en-us/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)
