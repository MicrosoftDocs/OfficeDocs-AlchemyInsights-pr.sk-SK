# <a name="change-the-domain-for-microsoft-365-group"></a>Zmena domény pre Microsoft 365 skupiny

Ak chcete zmeniť doménu pre skupinu Microsoft 365 počas vytvárania, postupujte podľa krokov v [Vyberte doménu použiť pri vytváraní Microsoft 365 skupiny](https://docs.microsoft.com/office365/admin/create-groups/choose-domain-to-create-groups).
  
Po Microsoft 365 skupina je už vytvorená, momentálne neexistuje žiadny spôsob, ako zmeniť doménu. Jediný spôsob, ako je vytvoriť novú skupinu Microsoft 365 a odstrániť staré, ak je to potrebné (ďalšie e-mailové adresy môžu byť pridané prostredníctvom [Exchange Admin Center](https://outlook.office365.com/ecp.aspx)).
