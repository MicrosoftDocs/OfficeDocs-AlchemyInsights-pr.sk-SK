# <a name="create-a-shared-mailbox"></a>Vytvorenie zdieľanej poštovej schránky

Zdieľaná poštová schránka umožňuje viac ako jednej osobe odosielať a prijímať e-maily pre alias, napríklad sales@contoso.com, bez priradenia inej licencie. 

Pozrite si článok [vytvorenie zdieľanej poštovej schránky](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) pre tipy a ďalšie informácie o používaní zdieľanej poštovej schránky.
