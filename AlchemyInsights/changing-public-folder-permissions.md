# <a name="changing-public-folder-permissions"></a>Zmeniť povolenia pre verejný priečinok

Vy a vaši používatelia môžete zmeniť povolenia pre verejný priečinok v programe Outlook. Pozri [nastaviť alebo zmeniť povolenia pre verejný priečinok](https://support.office.com/article/set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005). Správcovia môžu tiež zmeniť povolenia na výmenu admin center:
  
- Microsoft 365 admin Center, prejdite na **Admin centier** \> **výmena**.
    
- Vyberte **verejné priečinky**.
    
Odtiaľ môžete zmeniť povolenia pre jednotlivé verejné priečinky, priradenie povolenia používateľom skupiny zabezpečenia. Ak chcete zmeniť povolenia pre verejný priečinok, musíte byť vlastníkom to.
