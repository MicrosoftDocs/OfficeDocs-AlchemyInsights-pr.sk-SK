# <a name="migrating-to-microsoft-intune"></a>Prechod na Microsoft Intune

Ak sa pokúšate migrovať do intune skontrolovať [Intune migrácie Sprievodca](https://docs.microsoft.com/intune/fundamentals/migration-guide).

Pre podrobnejšie pokyny na migráciu z System Center Configuration Manager (a SCCM Hybrid) intune, reivew naše [hybridnej migrácie dokumentácie](https://docs.microsoft.com/sccm/mdm/deploy-use/migrate-hybridmdm-to-intunesa).
