# <a name="copy-and-paste-in-outlookcom"></a>Skopírujte a prilepte v službe Outlook.com

Nie sú kópie alebo prilepiť tlačidlá v službe Outlook.com, ale môžete použiť klávesové skratky:

**Na PC:** CTRL + C pre kopírovanie, CTRL + X Vystrihnúť a CTRL + V na prilepenie.

**Na Mac:** CMD + C pre kopírovanie, CMD + X Vystrihnúť a CMD + V na prilepenie.
