# <a name="create-and-use-a-shared-mailbox"></a>Vytvorenie a používanie zdieľanej poštovej schránky

Zdieľanej poštovej schránky umožňuje viacerým osobám, odosielanie a prijímanie e-mail pre alias, ako **sales@contoso.com**, bez priradenia ďalšiu licenciu.
  
[Vytvorenie zdieľanej poštovej schránky](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) vysvetľuje ako: 
  
- Vytvorenie zdieľanej poštovej schránky
    
- Poskytnite používateľom povolenia na používanie zdieľanej poštovej schránky
    
- Prístup k zdieľanej poštovej schránky v programe Outlook alebo v Outlook na webe
