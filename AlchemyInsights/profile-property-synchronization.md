# <a name="profile-property-synchronization"></a>Synchronizácia vlastností profilu

Nie je možné priradiť ďalšie vlastnosti pre import reklamy na synchronizáciu zo služby Azure Active Directory do aplikácie používateľského profilu. Importovanie reklamy je obmedzené na vopred nakonfigurovanú množinu vlastností, aby sa zaručil konzistentný výkon úlohy časovača.
  
[Ďalšie informácie o vlastnostiach profilu, ktoré sa synchronizujú](https://go.microsoft.com/fwlink/?linkid=875671)
