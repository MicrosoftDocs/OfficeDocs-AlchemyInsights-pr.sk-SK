# <a name="manage-who-can-create-microsoft-365-groups"></a>Správa osôb, ktoré môžu vytvárať skupiny microsoft 365

V predvolenom nastavení môže ktokoľvek vo vašej organizácii vytvárať skupiny služieb O365.
  
Ak to chcete zmeniť, môžete vytvoriť skupinu zabezpečenia pomocou prostredia PowerShell, ktorá obmedzuje vytváranie skupín na podmnožinu používateľov.
  
Postupujte podľa podrobných pokynov tu: [Správa, kto môže vytvoriť skupiny Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
