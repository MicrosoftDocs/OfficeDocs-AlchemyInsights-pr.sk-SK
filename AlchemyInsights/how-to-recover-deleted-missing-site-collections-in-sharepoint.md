# <a name="recover-missing-or-deleted-site-collections"></a>Obnoviť chýbajúce alebo vypúšťa kolekcií lokalít

Ako obnoviť chýbajúce alebo vypúšťa kolekcií lokalít.

[Obnovenie odstránených lokalít](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

[Obnovenie pomocou SPO PowerShell modul](https://support.office.com/article/Introduction-to-the-SharePoint-Online-Management-Shell-C16941C3-19B4-4710-8056-34C034493429)

Ak nemôžete nájsť lokalitu možno budete chcieť [vyhľadávanie denník auditu v Office 365 zabezpečenia &amp; centrum súladu](https://docs.microsoft.com/office365/securitycompliance/search-the-audit-log-in-security-and-compliance).

**Poznámka**: "Moderné" tímové lokality nie sú viditeľné v **klasickej** SharePoint admin UI. Zoznam lokalít, "moderné" môžete pristupovať z [nového Sharepoint admin center](https://docs.microsoft.com/sharepoint/get-started-new-admin-center).
