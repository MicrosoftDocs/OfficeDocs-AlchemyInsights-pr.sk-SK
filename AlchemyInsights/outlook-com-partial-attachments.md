# <a name="cant-preview-all-of-an-attachment"></a>Nie je možné zobraziť ukážku všetkých príloh

Sme si vedomí problému, kedy niektorí zákazníci nemôžu zobraziť ukážku všetkých príloh. Ako riešenie, prosím, otvorte prílohu v novom okne.
