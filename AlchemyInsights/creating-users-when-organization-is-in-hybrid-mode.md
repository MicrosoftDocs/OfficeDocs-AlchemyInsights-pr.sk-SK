# <a name="create-users-in-hybrid-deployments"></a>Vytvoriť používateľov v hybridné nasadenie

Ak vaša organizácia hybridné nasadenie lokálne prostredie a Office 365, musíte vytvoriť všetky používateľské objekty služby Active Directory lokálne. Môžete vytvoriť používateľské objekty vhodné organizačné jednotky (OU) tak ste synchronizovať do balíka Office 365 pre vašu organizáciu.
