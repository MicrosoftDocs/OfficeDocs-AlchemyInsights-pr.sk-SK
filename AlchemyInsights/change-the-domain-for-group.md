# <a name="change-the-domain-for-microsoft-365-group"></a>Zmena domény pre skupinu Microsoft 365

Ak chcete počas vytvárania zmeniť doménu pre skupinu Microsoft 365, postupujte podľa krokov v téme [Výber domény, ktorá sa má použiť pri vytváraní skupín microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/choose-domain-to-create-groups).

Po microsoft 365 skupina je už vytvorená, v súčasnosti neexistuje žiadny spôsob, ako zmeniť doménu. Jediný spôsob je vytvoriť novú skupinu Microsoft 365 a v prípade potreby odstrániť tú starú skupinu (ďalšie e-mailové adresy je možné pridať prostredníctvom [Centra spravovania pre Exchange](https://outlook.office365.com/ecp).
