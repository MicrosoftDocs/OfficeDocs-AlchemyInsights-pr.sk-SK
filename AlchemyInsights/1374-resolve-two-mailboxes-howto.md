# <a name="a-user-has-two-mailboxes"></a>Používateľ má dve poštové schránky

Hybridné prostredia, ktoré využívajú Azure Active Directory pripojenie (AAD pripojenie) alebo DirSync spôsobovať omylom používateľa mať dve poštové schránky: jeden v lokálnej organizácii a jedno v riešení cloud.
  
Ak chcete vyriešiť tento problém, pozri [Natrvalo vymazať predchádzajúce poštovej schránky Info](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).
