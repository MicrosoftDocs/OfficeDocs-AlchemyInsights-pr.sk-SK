[Zdroje na migráciu aplikácií do služby Azure Active Directory](https://docs.microsoft.com/azure/active-directory/manage-apps/migration-resources) – zdroje informácií, ktoré vám pomôžu migrovať Access a overovanie aplikácií do služby Azure Active Directory (Azure AD).

[Presunutie overovania aplikácie zo služby Active Directory Federation Services do služby Azure Active Directory](https://docs.microsoft.com/azure/active-directory/manage-apps/migrate-adfs-apps-to-azure) – vývojár zameral usmernenie na migráciu aplikácií zo služieb Active Directory Federation Services do služby Azure Active Directory.
