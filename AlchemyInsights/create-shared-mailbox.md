# <a name="create-a-shared-mailbox"></a>Vytvorenie zdieľanej poštovej schránky

Zdieľanej poštovej schránky umožňuje viacerým osobám, odosielanie a prijímanie e-mail pre alias, ako sales@contoso.com, bez priradenia ďalšiu licenciu. Na vytvorenie zdieľanej poštovej schránky, prejdite na [Zdieľané príručka e-mailu](https://portal.office.com/adminportal/home).
  
Nájdete v článku [vytvorenie zdieľanej poštovej schránky](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) pre tipy a ďalšie informácie o používaní zdieľanej poštovej schránky.
