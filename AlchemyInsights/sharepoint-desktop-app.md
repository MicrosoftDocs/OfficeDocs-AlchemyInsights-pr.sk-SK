# <a name="desktop-app-for-sharepoint"></a>Počítačová aplikácia pre SharePoint

Vaša organizácia je zaregistrovaná v programe Private Preview pre počítačovú aplikáciu pre SharePoint.

Spoločnosť Microsoft v súčasnosti nepodporuje počítačovú aplikáciu pre SharePoint (Súkromná Ukážka). Ak chcete získať pomoc s počítačovou aplikáciou pre SharePoint, použite zadanú aplikáciu Microsoft Private Preview Partners.
