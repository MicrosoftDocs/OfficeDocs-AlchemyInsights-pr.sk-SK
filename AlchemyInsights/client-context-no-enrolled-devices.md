# <a name="client-context---no-enrolled-devices"></a>Kontext klienta – žiadne zaregistrované zariadenia

Váš nájomník nemá žiadne zaregistrované zariadenia na správu mobilných zariadení (MDM) so službou Intune. Ak chcete svoje zariadenia zaregistrovať a spravovať prostredníctvom služby Intune MDM, prečítajte [si dokumentáciu na](https://docs.microsoft.com/intune/device-enrollment) požadovanej platforme. Ak používate tretiu stranu MDM alebo len funkcie ochrany aplikácií, nie je potrebné vykonať žiadnu akciu.
