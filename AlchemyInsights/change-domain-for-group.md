# <a name="change-the-domain-for-microsoft-365-group"></a>Zmena domény pre skupinu Microsoft 365

Ak chcete zmeniť doménu pre skupinu Microsoft 365 počas vytvárania, postupujte podľa krokov v téme [Výber domény, ktorá sa má použiť pri vytváraní skupín microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/choose-domain-to-create-groups).
  
Po microsoft 365 skupina je už vytvorená, v súčasnosti neexistuje žiadny spôsob, ako zmeniť doménu. Jediný spôsob, ako je vytvoriť novú skupinu Microsoft 365 a odstrániť staré v prípade potreby (ďalšie e-mailové adresy môžu byť pridané prostredníctvom [Exchange Admin Center](https://outlook.office365.com/ecp.aspx)).
