# <a name="microsoft-stream-licensing-overview"></a>Prehľad licencií na Microsoft Stream

Microsoft Stream je cloudová služba s licenciou na základe predplatného na používateľa. Je k dispozícii ako súčasť oprávnených predplatných na Microsoft 365. Informácie o každom type predplatného nájdete v časti ceny služieb Microsoft 365.

- [Prehľad licencií na Microsoft Stream](https://docs.microsoft.com/stream/license-overview)
