# <a name="learn-about-microsoft-365-groups"></a>Ďalšie informácie o skupinách Microsoft 365

Microsoft 365 skupiny je služba, ktorá pracuje s Office 365 nástroje, ktoré používate už, takže môžete spolupracovať s vašimi spoluhráčmi pri písaní dokumentov, vytváranie tabuliek, práca na projektové plány, plánovanie schôdzí, alebo odosielanie e-mailov. Ďalšie informácie sa nachádzajú v téme [informácie o skupinách spoločnosti Microsoft 365](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
