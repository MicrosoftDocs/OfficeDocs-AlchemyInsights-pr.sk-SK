# <a name="copy-and-paste-in-outlookcom"></a>Skopírujte a prilepte do Outlook.com

V Outlook.com nie sú tlačidlá kopírovať ani prilepiť, ale môžete použiť klávesové skratky:

**Na PC:** CTRL + C skopírovať, CTRL + X Vystrihnúť a CTRL + V Prilepiť.

**Na Macu:** CMD + C skopírovať, CMD + X znížiť, a CMD + V Vložiť.
