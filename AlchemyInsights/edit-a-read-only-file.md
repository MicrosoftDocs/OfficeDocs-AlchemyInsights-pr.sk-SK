# <a name="edit-a-read-only-file"></a>Upraviť súbor iba na čítanie

V knižniciach dokumentov, po kliknutí na názov dokumentu, Office program môže neočakávane otvoriť ako súbor iba na čítanie. Otvorte pre editáciu, kliknite na tlačidlo **Otvoriť** (tri bodky) a potom kliknite na tlačidlo **Upraviť**
