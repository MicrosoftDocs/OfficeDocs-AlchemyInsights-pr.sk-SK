# <a name="deploy-office-apps"></a>Nasadenie aplikácií balíka Office

Nasadenie aplikácií balíka Office na viacerých koncových používateľov prostredníctvom nástroja na nasadenie balíka Office, Microsoft Endpoint Configuration Manager, ako súčasť obraz disku operačného systému alebo v scenári zdieľaného počítača pomocou systému Windows Server s Remote Desktop Services (RDS) alebo terminálových služieb, zvážte použitie sprievodcu krok za krokom v **Microsoft 365 aplikácie pre Enterprise Deployment Advisor**.
  
Ak chcete začať, uistite sa, že ste prihlásený ako správca nájomníka alebo globálny správca, a potom kliknite sem: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
