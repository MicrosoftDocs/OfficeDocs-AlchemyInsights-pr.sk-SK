# <a name="set-up-microsoft-information-protection"></a>Nastavenie ochrany informácií spoločnosti Microsoft

Získajte prehľad o možnostiach, ktoré môžete použiť v stratégii ochrany informácií, aby ste si mohli byť istí, že vaše citlivé informácie sú chránené. Použite prístup k štyrom etapám životného cyklu, v ktorom zistíte, klasifikujú, chránite a sledujete citlivé informácie. Sprievodca nastavením ochrany informácií spoločnosti Microsoft poskytuje usmernenie na dokončenie každej z týchto fáz.
