# <a name="activating-windows-10-subscriptions"></a>Aktivácia predplatného systému Windows 10

Microsoft 365 Business, Microsoft 365 Education, Microsoft 365 a Enterprise poskytujú digitálne licencie pre Windows 10 ako súčasť predplatného. Ak chcete aktivovať predplatné, zariadenie už musí mať aktivovaný systém Windows 10 Pro (1703 alebo novší). Ak chcete zobraziť úplný zoznam požiadaviek na aktiváciu predplatného na Windows 10, pozrite si [požiadavky na aktiváciu predplatného na Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
