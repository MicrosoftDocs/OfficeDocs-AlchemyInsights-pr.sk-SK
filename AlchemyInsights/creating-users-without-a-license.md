# <a name="create-mailboxes-that-dont-require-licenses"></a>Vytvorenie poštových schránok, ktoré nevyžadujú licencie

- Ak potrebujete skupinovú e-mailovú adresu pre viacerých ľudí, pozrite si [vytvorenie zdieľanej poštovej schránky](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox).

- Ak chcete umožniť tímu ľudí spolupracovať a zdieľať e-maily a súbory, pozrite si [informácie o skupinách spoločnosti Microsoft 365](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
