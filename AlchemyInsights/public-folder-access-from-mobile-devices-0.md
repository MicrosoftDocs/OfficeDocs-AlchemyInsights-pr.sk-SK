# <a name="public-folder-access-from-mobile-devices"></a>Prístup k verejnému priečinku z mobilných zariadení

Verejné priečinky nie sú prístupné z mobilného zariadenia.
  
Ak chcete mať funkcie podobné na verejné priečinky a prístup z mobilných zariadení, nájdete [Ďalšie informácie o Office 365 skupiny](https://support.office.com/en-us/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) lepšou alternatívou.
