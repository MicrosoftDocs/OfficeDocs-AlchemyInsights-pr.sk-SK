# <a name="how-to-select-all-in-outlookcom"></a>Ako "vybrať všetko" v Outlook.com

Môžete vybrať iba všetky, ak sú všetky správy viditeľné. To je zabrániť náhodnému akcie na správy nevidíte.

**Ctrl + Click** vyberie viaceré jednotlivé správy.

**Shift + click** vyberie všetky správy medzi prvou a poslednou správou, ktorú vyberiete.
