# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>Previesť poštovej schránky používateľa poštovej schránky miestnosti alebo zariadenia

Môžete tiež previesť poštovej schránky z jednej poštovej schránky typu na iný, ale budete musieť vykonať tieto úlohy pomocou výmeny Online PowerShell. Pozrite sa na [prevod poštovej schránky](https://go.microsoft.com/fwlink/p/?LinkId=832875) pre viac informácií.
