# <a name="proxyaddress-incorrect"></a>ProxyAddress nesprávne

Keď objekt je synchronizované Azure AD, hodnoty, ktoré sú uvedené v atribúte proxyAddresses v službe Active Directory sú v porovnaní s Azure AD pravidlá a potom proxyAddresses atribút je vyplnený v Azure AD. Preto hodnoty proxyAddresses atribút objektu v službe Active Directory nemusia zhodovať hodnoty atribút proxyAddresses v Azure AD.
  
Ďalšie informácie o tom, ako je vyplnený proxyaddress vidieť [ako proxyAddress atribút je vyplnený v Azure AD](https://support.microsoft.com/en-us/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
