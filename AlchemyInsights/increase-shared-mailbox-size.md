# <a name="increase-the-size-of-a-shared-mailbox"></a>Zväčšiť veľkosť zdieľanej poštovej schránky

Môžete zväčšiť veľkosť zdieľanej poštovej schránky z predvolenej veľkosti 50 GB 100 GB priradením zdieľanej poštovej schránky Exchange Online P2 licencie (ktorá je zahrnutá do E3 a E5 licencie). Keď priradíte P2 licencie zdieľanej poštovej schránky v [aktívnych používateľov](https://portal.office.com/adminportal/home), veľkosť sa automaticky zvýši do 15 minút.
