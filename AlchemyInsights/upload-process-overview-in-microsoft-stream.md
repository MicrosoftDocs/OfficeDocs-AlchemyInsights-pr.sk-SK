# <a name="upload-process-overview-in-microsoft-stream"></a>Nahrať prehľad procesu v Microsoft Stream

Vždy, keď nahráte video, Microsoft Stream ho automaticky zakóduje do viacerých bitrate, takže video je možné vysielať pomocou adaptívneho dátového toku. Microsoft Stream optimalizuje dátový tok a kvalitu prehrávacieho videa na základe dostupnej šírky pásma diváka.

- [Nahrať prehľad procesu v Microsoft Stream](https://docs.microsoft.com/stream/upload-process-overview)
