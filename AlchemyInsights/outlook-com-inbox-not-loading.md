# <a name="inbox-not-loading-with-firefox"></a>Doručená pošta nenačítava s Firefoxom

Niektoré doplnky v prehliadači Firefox môže zabrániť nakladanie doručenej pošty služby Outlook.com.
  
Navštívte Firefox support naučiť ako [vypnúť alebo odstrániť Doplnky](https://support.mozilla.org/kb/disable-or-remove-add-ons).
