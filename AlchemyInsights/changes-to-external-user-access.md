# <a name="changes-to-external-user-access"></a>Zmeny prístupu externých používateľov

Od 23. marca 2018, Externí používatelia už nebudú môcť získať prístup k obsahu, ktorý sa zdieľa s každým, všetkými overenými používateľmi alebo všetkými používateľmi formulárov, dokonca aj vtedy, ak boli pozvaní alebo pridaní do autorizovaných skupín. Externí používatelia budú môcť získať prístup k obsahu, ktorý sa zdieľa s používateľom, alebo so skupinami, ku ktorým daný používateľ patrí. Ak chcete zachovať existujúce nastavenia, prečítajte si tému: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
