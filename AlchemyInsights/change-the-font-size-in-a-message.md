# <a name="change-the-font-size-in-a-message"></a>Zmena veľkosti písma v správe

1. Ak ste správu už zadali, vyberte text.
2. Vyberte ikonu  **formátovať** pod správou (je to prvá ikona).
3. Vyberte ikonu  **veľkosť písma**  nad správou (je to siedma ikona).
4. Vyberte veľkosť.
