# <a name="public-folder-access-from-mobile-devices"></a>Prístup k verejným priečinkom z mobilných zariadení

Verejné priečinky nie sú prístupné z mobilných zariadení.
  
Ak chcete mať podobné funkcie ako verejné priečinky a máte k nim prístup z mobilných zariadení, pozrite si tému [informácie o skupinách v službe Microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) , ktoré umožňujú lepšiu alternatívu.
