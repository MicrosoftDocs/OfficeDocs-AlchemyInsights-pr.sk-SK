# <a name="give-users-access-to-sharepoint-site"></a>Poskytnúť používateľom prístup k lokalite SharePoint

Funkcia žiadosť o prístup umožňuje ľuďom požiadať o prístup k obsahu, ktorý v súčasnosti nemajú povolenie na zobrazenie. Ako vlastník lokality môžete nakonfigurovať funkciu, ktorá vám pošle poštu, keď niekto požiada o prístup na lokalitu. Ďalšie informácie nájdete [v téme Nastavenie a správa žiadostí o prístup](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3).
