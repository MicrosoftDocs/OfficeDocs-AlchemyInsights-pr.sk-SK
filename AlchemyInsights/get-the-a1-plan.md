# <a name="get-the-a1-plan"></a>Získať plán A1

Ak váš inštitút už je kvalifikovaný pre Microsoft 365 Education, môžete pridať predplatné A1 z **Centra spravovania služby Microsoft 365**. 

1. Prejdite na položku **Fakturácia > [Produkty a služby](https://go.microsoft.com/fwlink/p/?linkid=868433)**.

2. Pomocou vyhľadávacieho poľa vyhľadajte výraz „A1“.

3. Vyberte počet používateľov, ktorí budú používať plán A1.

Ak váš inštitút ešte nenainštaloval program Microsoft 365 Education, ale chcete zdarma pridať Office 365, môžete [začať](https://www.microsoft.com/education/products/office) tak, že nám poskytnete svoj e-mail.
