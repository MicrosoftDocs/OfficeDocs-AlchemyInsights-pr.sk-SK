# <a name="unable-to-print"></a>Nie je možné tlačiť

Sme si vedomí problému, kedy niektorí používatelia nemôžu tlačiť pri používaní prehliadačov Safari alebo Edge. Aj keď pracujeme na opravu pre to, prosím, použite iný prehliadač. Ospravedlňujeme sa za nepríjemnosti.
