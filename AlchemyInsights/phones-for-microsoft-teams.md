# <a name="are-you-looking-for-phones-that-are-compatible-with-teams"></a>Hľadáte telefóny, ktoré sú kompatibilné s tímami?

Ako správca [spravujete všetky zariadenia](https://docs.microsoft.com/microsoftteams/device-management) používané s tímami vo vašej organizácii z Microsoft 365 admin Center. Ďalšie informácie: 

- Skontrolujte zoznam [telefónov Microsoft teams](https://docs.microsoft.com/microsoftteams/phones-for-teams). 
- Skontrolujte zoznam [aktuálne CERTIFIKOVANÝCH IP telefónov](https://docs.microsoft.com/microsoftteams/teams-ip-phones#currently-certified-ip-phones). 
- Skontrolujte zoznam [zariadení so známymi obmedzeniami a problémami](https://support.office.com/article/control-calls-using-a-headset-in-teams-65d6e104-444d-4013-b8c2-f11317dd69a8).
