# <a name="create-users-in-hybrid-deployments"></a>Vytvorenie používateľov v hybridnom nasadení

Keď je vaša organizácia hybridným nasadením lokálneho prostredia a Microsoft 365, musíte vytvoriť všetky objekty používateľa v lokálnej službe Active Directory. Vytvoríte objekty používateľa v príslušnej organizačnej jednotky (OU) tak, aby boli synchronizované do vašej organizácie.
