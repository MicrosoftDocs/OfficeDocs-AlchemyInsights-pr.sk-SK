# <a name="print-results-in-a-summary-of-responses"></a>Tlač výsledkov v súhrne odpovedí

Výsledky v [službe Microsoft Forms môžete vytlačiť](https://support.office.com/article/print-a-form-22100b98-ba3c-41c1-9513-f76caca664fc) ako súhrn odpovedí alebo jednotlivca. 

Nie je možné tlačiť formuláre bez odpovedí, ale môžete požiadať o túto funkciu alebo hlasovať na [lokalite Microsoft Forms uservoice](https://microsoftforms.uservoice.com/forums/386451-welcome-to-microsoft-forms-suggestion-box).
