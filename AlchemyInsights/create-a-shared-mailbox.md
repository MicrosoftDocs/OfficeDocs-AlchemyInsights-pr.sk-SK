# <a name="create-a-shared-mailbox"></a>Vytvorenie zdieľanej poštovej schránky

Zdieľanej poštovej schránky umožňuje viacerým osobám, odosielanie a prijímanie e-mail pre alias, ako **sales@contoso.com**, bez priradenia ďalšiu licenciu.
  
Ďalšie informácie v téme [vytvorenie zdieľanej poštovej schránky](https://support.office.com/client/871a246d-3acd-4bba-948e-5de8be0544c9) .
