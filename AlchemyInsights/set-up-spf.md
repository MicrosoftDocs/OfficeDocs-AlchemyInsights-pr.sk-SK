# <a name="set-up-spf"></a>Nastavenie SPF

Aktualizujte svoje DNS záznamy, aby ste mohli na overenie odchádzajúcich e-mailov odoslaných z vlastnej domény použiť rámec politiky odosielateľa (SPF) s vlastnou doménou. Týmto spôsobom zabránite iným používateľom v pošte spoofing, akoby bol odoslaný z vašej domény.
  
- [Nastavenie SPF na prevenciu spoofing](https://docs.microsoft.com/microsoft-365/security/office-365-security/set-up-spf-in-office-365-to-help-prevent-spoofing)

- [Pomoc pri nastavovaní všetkých DNS záznamov](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider)
