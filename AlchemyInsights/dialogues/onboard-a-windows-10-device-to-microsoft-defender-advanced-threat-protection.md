# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>Integrovanie zariadenia s Windowsom 10 do rozšírenej ochrany pred hrozbami pre Microsoft Defender (MDATP)

MDATP podporuje nasledujúce nástroje a metódy nasadenia pre Windows 10:

- Skupinová politika
- Microsoft Endpoint Configuration Manager
- Správa mobilných zariadení vrátane služby Microsoft Intune
- Lokálne skripty

Ďalšie informácie nájdete v téme [Nástroje a metódy pre zariadenia s Windowsom 10](https://go.microsoft.com/fwlink/?linkid=2143460).
