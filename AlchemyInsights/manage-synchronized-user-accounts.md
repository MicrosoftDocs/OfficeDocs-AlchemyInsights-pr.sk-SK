# <a name="manage-synchronized-user-accounts"></a>Správa synchronizovaných používateľských kont

Zvyčajne používate Microsoft 365 admin Center pridať a zmeniť používateľské kontá. Však nemôžete použiť, ak adresár sychronization alebo SSO sú povolené. Namiesto toho spravovať používateľov pomocou prostredia Windows PowerShell, ako je opísané v [konfigurovať vlastnosti používateľského konta s Office 365 PowerShell](https://docs.microsoft.com/office365/enterprise/powershell/configure-user-account-properties-with-office-365-powershell ).
