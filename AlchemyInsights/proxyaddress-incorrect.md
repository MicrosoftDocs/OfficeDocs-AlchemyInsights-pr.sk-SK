# <a name="proxyaddress-incorrect"></a>ProxyAddress nesprávne

Keď sa objekt synchronizuje so službou Azure AD, hodnoty, ktoré sú uvedené v atribúte proxyAddresses v službe Active Directory, sa porovnávajú s pravidlami Azure AD a potom je atribút proxyAddresses osídlený v službe Azure AD. Preto hodnoty atribútu proxyAddresses pre objekt v službe Active Directory sa nemusia zhodovať s hodnotami atribútu proxyAddresses v službe Azure AD.
  
Ďalšie informácie o tom, ako je proxyAddress vyplnený, nájdete [v téme ako sa atribút proxyAddress vyplní v službe Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
