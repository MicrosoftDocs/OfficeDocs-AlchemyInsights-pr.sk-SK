# <a name="copy-and-paste-in-outlookcom"></a>Kopírovanie a prilepenie v Outlook.com

V programe Outlook.com nie sú k dispozícii žiadne tlačidlá na kopírovanie ani prilepenie, ale môžete použiť klávesové skratky:

**V počítači:** Stlačením kombinácie klávesov CTRL + C skopírujte, stlačte kombináciu klávesov CTRL + X na vystrihnutie a stlačením kombinácie klávesov CTRL + V prilepte.

**V Macu:** CMD + C na kopírovanie, CMD + X na vystrihnutie a CMD + V na prilepenie.
