# <a name="cant-download-files-in-chrome"></a>Nemôžete sťahovať súbory v prehliadači Chrome

**Problém** Sme si vedomí problému, ktorý bráni niektorí zákazníci budú môcť sťahovať súbory pri používaní prehliadača Google Chrome. 

**Riešenie** Použite [Microsoft Edge](https://www.microsoft.com/en-gb/windows/microsoft-edge) alebo iný webový prehliadač.
Ďalšie informácie o tom, ako [sťahovať súbory a Outlook.com príloh](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab).
