# <a name="proxyaddress-incorrect"></a>ProxyAddress nesprávne

Keď objekt je synchronizovaný Azure AD, hodnoty, ktoré sú špecifikované v atribúte proxyAddresses v službe Active Directory sú porovnané s pravidlami Azure AD a potom proxyAddresses atribút je vyplnený v Azure AD. Preto hodnoty proxyAddresses atribút objektu v službe Active Directory nemusia byť rovnaké ako hodnoty proxyAddresses atribút Azure AD.
  
Ďalšie informácie o tom, ako proxyAddress je vyplnený, pozrite [sa, ako proxyAddress atribút je vyplnený v Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
