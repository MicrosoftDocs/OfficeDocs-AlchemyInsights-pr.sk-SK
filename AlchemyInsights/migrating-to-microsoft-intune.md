# <a name="migrating-to-microsoft-intune"></a>Migrácia do služby Microsoft Intune

Ak sa pokúšate migrovať do služby Intune, prečítajte si [príručku na migráciu služby Intune](https://docs.microsoft.com/intune/fundamentals/migration-guide).

Ďalšie konkrétne pokyny na migráciu zo služby System Center Configuration Manager (a SCCM Hybrid) do služby Intune, Reivew našej [hybridnej migrácie](https://docs.microsoft.com/sccm/mdm/deploy-use/migrate-hybridmdm-to-intunesa).
