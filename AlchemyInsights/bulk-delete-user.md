# <a name="bulk-delete-user"></a>Používateľ hromadného odstránenia

Veľký počet používateľov zo skupiny môžete odstrániť pomocou súboru s hodnotami oddelenými čiarkou (CSV) na hromadné odstránenie používateľov. Ak chcete odstrániť hromadných používateľov pomocou portálu Azure AD, pozrite si [tému Hromadné odstránenie používateľov v službe Azure Active Directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/users-bulk-delete).
